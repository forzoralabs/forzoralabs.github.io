Privacy Policy

This Privacy Policy explains how ForzoraLabs handles information in relation to the NetPulse application. NetPulse is a free app designed to test and display internet performance, including download speed, upload speed, and ping.

NetPulse does not collect, store, sell, or share personal data from users.

1. Information We Collect

NetPulse does not collect personal information from users.

The app does not require account creation, does not request names, email addresses, phone numbers, identification details, payment information, or location data, and does not store personal usage data on external servers.

NetPulse is built to perform internet speed measurement only. Its main purpose is to test network performance by checking values such as download speed, upload speed, and ping on the user's device.

2. Permission Used

NetPulse only requests one permission:

  • Post Notifications — this permission is used only to show notifications related to the operation of the app where needed.

This permission is not used to collect user information. It is only used so the app can properly display notifications on supported Android versions.

3. How NetPulse Works

NetPulse works by performing internet speed tests on the device. During a test, the app measures:

  • Download speed — how fast data is received from the internet
  • Upload speed — how fast data is sent to the internet
  • Ping — the response time between the device and a network server

These measurements are used only to display speed test results to the user inside the app. NetPulse does not create user profiles and does not track users for advertising or personal analytics.

4. Data Sharing

Since NetPulse does not collect personal user data, it does not sell or share personal information with third parties.

ForzoraLabs does not use the app to trade user information, and no personal data is processed for marketing purposes.

5. Children's Privacy

NetPulse does not knowingly collect personal information from children or from any users. Since the app does not collect user data, there is no intentional collection of children's personal data.

6. Changes to This Policy

ForzoraLabs may update this Privacy Policy from time to time if the app changes in the future. Any updates will be posted on this page with a revised effective date.
